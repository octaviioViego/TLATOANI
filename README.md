# TLATOANI
Aquí tienes una versión mejor redactada y más fluida: El proyecto “TLATOANI” es una iniciativa de procesamiento de lenguaje natural basada en una red neuronal diseñada para detectar el humor en textos. Utiliza el análisis de humor a partir de anotaciones humanas. Este trabajo se desarrolla como proyecto final para la materia de Redes Neuronales.
